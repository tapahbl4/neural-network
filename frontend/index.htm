This site is a visualization of the work of a neural network.
Developed for the thesis project Dmitry S. Marchenko, 4th year student of the BV group, specialty "Informatics", the Faculty of Mathematics and Natural Sciences, Mogilev State University named after Kuleshov

The main characteristics of the neural network
Characteristic Value
Neural Network Architecture Multilayer Perceptron
Learning Algorithm Method of back propagation error
Activation function The sigmoidal function with an interval of values (-1; 1)
Input data Time series of dollar exchange rate increments
Output data The increment relative to the last dollar value
Data source The values of the stock exchange, received via the API NBRB site
Dmitry Marchenko, 2017
